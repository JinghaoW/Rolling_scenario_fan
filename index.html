Methodology

1Read data: inflow (regulated + unregulated), load, wind production
2First stage: day-ahead dispatch with current reservoir state
3Second stage: K scenarios × W wind traces run in parallel for remaining horizon
4Extract Benders cuts: π = ∂(future cost) / ∂(reservoir volume)
5Iterate until reservoir convergence < 0.001
6Advance 1 day, repeat for 364 days

Objective Function — First Stage

min Σt ( Cshed·rt + Cbyp·qbt + Cspl·qst + Cwind·wct ) + α

Benders Cut

α ≥ Σk [ ObjkRT + Σm λkm · (Vm,1 − V̄km) ]
λ = marginal water value (dual of reservoir balance).
